# ⚙️ Installation

## Clone Repository

```bash
git clone https://github.com/priyanshichouhan07/Fasal_AI.git
```

```bash
cd Fasal_AI
```

---

## Install Dependencies

```bash
pip install -r requirements.txt
```

---

## Create `.env` File

```env
GEMINI_API_KEY=your_api_key

AGMARKNET_API_KEY=your_api_key
```

> ⚠️ Never upload your `.env` file or secret API keys to GitHub.

---

## Run Server

```bash
uvicorn main:app --reload
```

Server runs at:

```bash
http://127.0.0.1:8000
```
